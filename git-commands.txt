# Git Push Commands

## Main Repository
git remote remove origin
git remote add origin https://github.com/lovevideogames25-ui/ChatHub.git
git push -u origin main

## Test Repository  
git remote remove origin
git remote add origin https://github.com/lovevideogames25-ui/CHATHUB-TEST-WEB.git
git push -u origin main

## Check Remote
git remote -v

## Complete Workflow
git add .
git commit -m "Your commit message"
git pull origin main
git push -u origin main

## Quick Commands
git add .
git commit -m "Update"
git push

## Force Push (if needed)
git push -f origin main

## Push to Test Repository
git remote remove origin
git remote add origin https://github.com/lovevideogames25-ui/CHATHUB-TEST-WEB.git
git add .
git commit -m "Update test"
git push -u origin main

## Push to Main Repository
git remote remove origin
git remote add origin https://github.com/lovevideogames25-ui/ChatHub.git
git add .
git commit -m "Update main"
git push -u origin main
